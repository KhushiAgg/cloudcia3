Data Visualization & Analytical Tools in Different Cloud Platforms

IBM Cloud

IBM Cognos Analytics:

  • IBM Cognos Analytics is a comprehensive business intelligence and data visualization tool that allows users to create interactive dashboards, reports, and visualizations.
  • It supports a wide range of data sources and provides advanced analytics capabilities, including machine learning, to help users gain insights from their data.
  • IBM Cognos Analytics is designed for organizations that require robust and scalable analytics solutions.

    • IBM Watson Studio:

  • IBM Watson Studio is a data science and machine learning platform that also offers data visualization capabilities.
  • Users can leverage Watson Studio to prepare and analyze data, create visualizations, and build machine learning models all in one platform.
  • It's suitable for data scientists and analysts who need a comprehensive environment for data exploration, modeling, and visualization.

    • IBM DataRefinery (formerly known as DataWorks):

  • IBM DataRefinery is a cloud-based data preparation and data wrangling tool that helps users clean, shape, and enrich their data for analysis.
  • While it primarily focuses on data preparation, it also offers basic data visualization features to visualize data during the data cleansing and transformation process.
  • DataRefinery is a valuable tool for data engineers and analysts who need to ensure data quality and consistency before performing in-depth analysis.


    • Google Cloud Platform

    Google Data Studio:

  • Google Data Studio is a free and user-friendly data visualization tool that allows you to create interactive and shareable reports and dashboards.
  • It integrates seamlessly with other GCP services, as well as various data sources, making it a versatile choice for data visualization.
  • Data Studio is suitable for businesses and individuals looking for a straightforward tool to present data insights.

    • BigQuery:

  • While primarily known as a data warehouse, Google BigQuery includes data visualization capabilities. You can run SQL queries in BigQuery and create visualizations directly within the platform.
  • It's particularly beneficial for analysts and data engineers who want to analyze large datasets and generate visual insights without the need for separate tools.
  • BigQuery is ideal for organizations looking for an all-in-one solution for data storage, analysis, and visualization.

    • Looker:

  • Looker is a data exploration and business intelligence platform that is now part of Google Cloud.
  • It allows users to create interactive dashboards and reports, providing a holistic view of your data.
  • Looker supports data preparation, transformation, and sharing of insights, making it a comprehensive solution for data analysis.


    • Oracle Cloud

    Oracle Analytics Cloud (OAC):

  • Oracle Analytics Cloud is a comprehensive, cloud-based platform that provides self-service analytics and data visualization capabilities. It allows users to create interactive dashboards, reports, and data visualizations.
  • OAC supports a wide range of data sources, including Oracle databases, data warehouses, and third-party sources, making it suitable for organizations with diverse data requirements.
  • The platform also offers advanced features such as machine learning integration, natural language processing, and predictive analytics.

    • Oracle Data Visualization Desktop (DVD):

  • Oracle Data Visualization Desktop is a client-based data visualization tool that enables users to explore and visualize data on their local machines.
  • With DVD, users can connect to various data sources, create visualizations, and then share their work with others. It's suitable for individuals or teams who want to perform data analysis and create compelling visuals.
  • It supports data preparation, transformation, and blending to help users get the most out of their data.

    • Oracle Autonomous Database:

  • Oracle Autonomous Database is a cloud-native, self-driving database service. While it's primarily a database service, it offers built-in analytics and data visualization capabilities.
  • Users can run SQL queries and create visualizations directly within the Autonomous Database, making it a suitable choice for organizations that prefer a unified database and analytics platform.
  • Oracle's Autonomous Database can automatically optimize performance, security, and scalability, allowing users to focus on data analysis and insights.
